# Dino Stars PWA checklist

## Android / Chrome
- Open the live URL.
- Confirm it loads without browser errors.
- Open Parent Setup.
- Confirm Install button appears where supported.
- Install the app.
- Open from home screen.
- Confirm it opens in standalone/app-like mode.
- Complete a task and confirm the egg becomes a star.
- Refresh/reopen and confirm the star remains.

## iPhone / iPad Safari
- Open the live URL in Safari.
- Tap Share.
- Tap Add to Home Screen.
- Open Dino Stars from the home screen.
- Confirm app-like view, icon and name.

## Offline behaviour
- Open app once online.
- Turn off internet.
- Open app again.
- Confirm shell/offline message appears.
- Confirm task completion is not silently saved offline.

## Multi-device sync
- Open the app on two devices.
- Complete one George task on device A.
- Refresh device B.
- Confirm the same task is complete.
- Undo from Parent Setup.
- Refresh device A.
- Confirm the task is undone.

## Admin checks
- Initial PIN 2468 works.
- Change reward text.
- Change weekly target.
- Change George's daily task count.
- View activity log and confirm date/time/action details are visible.
